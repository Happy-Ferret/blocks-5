// §de: for German
// §fr: for French
// §es: for Spanish
//
// The paragraph character (¶) can be used to insert newlines!

§en:<h>The Level Editor</h>
With the level editor, you can create your own levels.
A level consists of two layers, on which tiles are placed, and of a number of objects. Tiles are used for ground, holes oder (destroyable) walls. Tiles can't move. Objects are, for example, blocks, diamonds, grass, players or enemies.

At the screen's bottom, you can choose the layer you want to work on. It is important to understand how the layers work. Layer 0 is for ground tiles (like lawn, sand or holes). A wall placed on layer 0 won't act like a wall: the player can walk trough. A "real" wall always has to be placed on layer 1!

You also see 5 categories: Tiles, Objects, Switches, Rails and Electronics. Here you can select what tile/object you want to place in the level. Simply click at one of these categories and choose the desired tile/object.
// --------------------------------------------------------------------------------
§de:<h>Der Level-Editor</h>
Mit dem Level-Editor kannst du eigene Levels erzeugen.
Levels bestehen aus zwei Ebenen (Layer), auf denen so genannte Tiles platziert werden, und aus einer Reihe von Objekten. Tiles sind Boden, Löcher oder (zerstörbare) Mauern. Sie können sich nicht bewegen. Objekte sind beispielsweise Blöcke, Diamanten, Gras, Spielfiguren oder Gegner.

Am unteren Bildschirmrand kannst du einstellen, auf welchem Layer du arbeiten möchtest. Es ist wichtig, die Layer genau zu verstehen. Auf Layer 0 befinden sich die Boden-Tiles (z.B. Rasen, Sand oder auch Löcher). Durch eine Mauer auf Layer 0 kann der Spieler einfach hindurch laufen. Eine "echte" Mauer muss sich immer auf Layer 1 befinden!

Unten gibt es 5 Kategorien: Tiles, Objekte, Schalter, Schienen und Elektronik. Hier kannst du wählen, welches Tile/Objekt du im Level platzieren willst. Klicke dazu eine der Kategorien an, und wähle dann das gewünschte Tile oder Objekt.

§en:<h>The Tools</h>
Pen: Draws the selected tile/object. Use the right mouse button to erase. When drawing tiles, make sure that you have selected the right layer. Objects do not belong to a certain layer. If you want to stack objects (for example a laser on a platform), hold the Shift key while drawing.

Modify: Use this tool to modify objects. Some objects can be modified in two ways. For the second one, hold the Shift key while clicking. Example: You can change the color of a cannon, but you can also turn the cannon (by holding Shift). The tool also allows you to change the text of a hint or a teleporter's target position (click and drag).

Rectangle: Draws a rectangle using the selected tile/object. Press the right mouse button to erase the selected area.

Transition: Helps you drawing transitions between two types of ground on layer 0 (like lawn and sand).
// --------------------------------------------------------------------------------
§de:<h>Die Werkzeuge</h>
Zeichenstift: Zeichnet das gewählte Tile/Objekt. Die rechte Maustaste dient als Radiergummi. Achte bei Tiles darauf, dass du sie auf den richtigen Layer setzt! Objekte sind hingegen keinem Layer zugeordnet. Um mehrere Objekte übereinander zu setzen (z.B. einen Laser auf eine Plattform), halte Shift gedrückt.

Modifizieren: Hiermit kannst du Objekte verändern. Klicke das zu verändernde Objekt dazu einfach an. Manche Objekte lassen sich auf zwei Arten verändern. Halte dazu gleichzeitig die Shift-Taste gedrückt. Beispiel: Bei Kanonen kannst du durch normales Modifizieren die Farbe ändern. Beim Modifizieren mit Shift dreht sich die Kanone. Bei Hinweiszetteln kannst du durch Modifizieren den Text verändern und bei Teleportern durch Klicken und Ziehen den Zielort.

Rechteck: Zeichnet ein Rechteck mit dem gewählten Tile/Objekt. Mit der rechten Maustaste wird der entsprechende Bereich geleert.

Übergang: Hilft dir beim Zeichnen von Übergängen zwischen zwei Bodentypen auf Layer 0 (zum Beispiel Rasen und Sand).

§en:<h>Encrypted Skins</h>
Skin zip archives may be encrypted so that their content can't be accessed by everyone. Virtually all zip tools support encryption. If you decide to encrypt your skin archive, you also have to include an unencrypted(!) file named "password.txt" containing the encrypted password. Use the tool "pwencrypt.exe", which resides in the same directory as "blocks5.exe" to encrypt the password.

<h>Custom Music</h>
In the "Settings" window, you can enter the name of a music file that the user should hear while playing the level. The file must be encoded in the OGG file format. Codecs can be found on the internet using a search engine. Music files have to be placed inside the "levels" directory.
// --------------------------------------------------------------------------------
§de:<h>Skins mit Passwort</h>
Skin-Zip-Archive können auch verschlüsselt gespeichert werden, damit die Grafiken nicht für jeden zugänglich sind. Praktisch alle Programme zum Erzeugen von Zip-Archiven unterstützen auch Verschlüsselung. Wenn das Archiv verschlüsselt wird, muss sich darin ebenfalls eine unverschlüsselte(!) Datei namens "password.txt" befinden. In diese Datei wird das zur Entschlüsselung benötigte Passwort verschlüsselt gespeichert. Zur Verschlüsselung des Passworts benutzt du das Programm "pwencrypt.exe", das sich im selben Verzeichnis befindet wie "blocks5.exe".

<h>Eigene Musikstücke</h>
